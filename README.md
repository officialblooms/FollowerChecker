# About

Amateur coding project that reads my Instagram's followers and following list and prints out a list of people that are not following me back and I am not following back.

# Installation

Clone this project into your local machine by doing:
```bash
git clone https://github.com/officialblooms/FollowerChecker
```

Make sure that [Java](https://www.java.com/download/ie_manual.jsp) and [OpenJDK](https://openjdk.org/install/) are installed on your machine, as well as [Apache Maven](https://maven.apache.org/install.html).

Drag your ```followers_1.json``` and ```following.json``` files into the directory, run the ```main.java``` file and you should be good to go!
